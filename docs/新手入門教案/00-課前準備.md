## 安裝清單

| 工具 | 版本 | 推薦安裝方式（macOS） | Windows WSL 注意事項 |
|------|------|----------------------|---------------------|
| Node.js 20 | 20.x LTS | [nvm](https://github.com/nvm-sh/nvm)：`nvm install 20` | WSL2 中同樣使用 nvm 安裝 |
| Docker Desktop | 最新版 | [Docker Desktop for Mac](https://www.docker.com/products/docker-desktop/) | 安裝 Docker Desktop for Windows 並啟用 WSL2 後端整合 |
| Git | 最新版 | `brew install git` 或 Xcode Command Line Tools | WSL2 內建或 `sudo apt install git` |
| IDE | — | **VS Code** + ESLint + Prettier | 可搭配 Remote - WSL 擴充 |

## 驗證指令

```bash
# 確認 Node.js 版本
node -v                          # 應顯示 v20.x

# 確認 npm 可用
npm -v

# 確認 Docker Compose 可用
docker compose version

# Clone 專案並安裝依賴
git clone <本 Repo URL> && cd nodejs-nestjs-sample
npm install                      # 確認能下載依賴
```

## 踩坑提示

| 現象 | 原因 | 解法 |
|------|------|------|
| `docker: command not found` | Docker Desktop 未啟動或未安裝 | macOS / Windows 需先啟動 Docker Desktop 應用程式 |
| npm install 卡住或逾時 | 公司網路可能封鎖 npm registry | 設定 npm proxy 或切換至個人網路 |
| `node -v` 顯示非 20 | 系統有多版本 Node.js | 用 nvm `nvm use 20` 或設定 `.nvmrc` |
